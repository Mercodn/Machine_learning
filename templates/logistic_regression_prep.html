Logistic Regression - Data Preparation

Data Loading

The dataset is loaded from a CSV file containing email features and spam labels.

Feature Selection

All four features (length, num_exclamation, num_dollar, num_uppercase) are selected as input variables, and the 'label' column is used as the target variable.

Train-Test Split

The data is split into training (80%) and testing (20%) sets using stratified sampling to maintain class distribution.

Model Training

A Logistic Regression model is trained on the training data to learn the relationship between email features and spam classification.
