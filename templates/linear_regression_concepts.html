Linear Regression - Basic Concepts

What is Linear Regression?

Linear regression is a supervised machine learning algorithm used to predict a continuous target variable based on one or more predictor variables. It assumes a linear relationship between the input variables (X) and the output variable (Y).

Independent and Dependent Variables
  • Independent Variables (Features): The input variables used to make predictions (e.g., age, BMI, smoking status)
  • Dependent Variable (Target): The output variable we want to predict (e.g., insurance charges)
Training Data

Training data consists of historical observations where we know both the input features and the corresponding target values. The algorithm learns the relationship between inputs and outputs from this data.

Regression Line

The regression line is represented by the equation: Y = mX + b

  • Y: Predicted value
  • X: Input feature
  • m: Slope (coefficient)
  • b: Intercept (constant)
Slope and Intercept
  • Slope (m): Indicates how much Y changes for a unit change in X
  • Intercept (b): The value of Y when X is zero
Example of Application

In our insurance dataset, we can predict charges based on age. For example, the model might learn that insurance charges increase by $200 for each additional year of age, with a base charge of $5,000 for a newborn.
