📈 Logistic Regression - Basic Concepts

Understanding binary classification with logistic regression

What is Logistic Regression?

Logistic Regression is a supervised learning algorithm used for binary classification—predicting which of two classes an instance belongs to. Unlike linear regression that predicts continuous values, logistic regression predicts probabilities between 0 and 1.

Binary Classification

Logistic regression answers yes/no questions:

  • Will the customer make a purchase? (Yes/No)
  • Is this email spam? (Yes/No)
  • Will the patient develop diabetes? (Yes/No)
  • Is this a cat in the image? (Yes/No)

The Logistic Function

p = 1 / (1 + e^(-z))

Where:

z = β₀ + β₁x₁ + β₂x₂ + ... + βₙxₙ

The logistic function produces an S-shaped curve that maps any input to a probability between 0 and 1.

Key Components

Independent Variables (Features)

Input data used to make predictions. Can be multiple variables.

In our model: Age, income, web visits, time on site, previous purchases, discount usage

Dependent Variable (Target)

Binary outcome: 0 (No) or 1 (Yes), or probability between 0 and 1.

In our model: 1 = Customer makes purchase, 0 = No purchase

Probability Threshold

Typically set at 0.5:

  • If P(event) ≥ 0.5 → Predict Class 1 (Yes)
  • If P(event) < 0.5 → Predict Class 0 (No)

How Logistic Regression Works

  1. Collect Labeled Data: Gather examples where outcome is known (0 or 1)
  2. Train the Model: Use optimization to fit the logistic curve
  3. Learn Coefficients: Determine weights for each feature
  4. Make Predictions: Output probabilities for new instances
  5. Apply Threshold: Convert probabilities to class predictions

Advantages of Logistic Regression

  • Simple and interpretable
  • Fast to train and predict
  • Provides probability estimates
  • Works well with linearly separable data
  • Less prone to overfitting

Applications

  • Medical diagnosis (disease/no disease)
  • Email spam filtering
  • Customer churn prediction
  • Credit approval decisions
  • Customer purchase prediction
