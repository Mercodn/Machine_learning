📊 Linear Regression - Basic Concepts

Understanding the fundamentals of linear regression

What is Linear Regression?

Linear Regression is a supervised learning algorithm that models the linear relationship between input variables (features) and a continuous output variable (target). It finds the best-fitting straight line through the data points.

The Regression Line Equation

y = mx + b

Or in ML notation:

ŷ = β₀ + β₁x

Key Components

Independent Variable (x)

The input feature used to make predictions. Also called predictor or feature.

Example: Study hours

Dependent Variable (y)

The output or target value we want to predict. Also called response or target.

Example: Final grade

Slope (β₁ or m)

Represents the change in the dependent variable for each unit change in the independent variable.

Interpretation: For every additional hour of study, the grade increases by [slope] points.

Intercept (β₀ or b)

The y-value when x equals zero. Represents the predicted value with no input.

Interpretation: Without any study hours, the predicted grade is [intercept].

How Linear Regression Works

  1. Collect Training Data: Gather examples with known inputs and outputs
  2. Find the Best Line: Use optimization to minimize prediction errors (residuals)
  3. Calculate Coefficients: Determine the slope and intercept values
  4. Make Predictions: Use the equation to predict new values

Assumptions of Linear Regression

  • Linear relationship between variables
  • Independent observations
  • Constant variance (homoscedasticity)
  • Normally distributed errors
