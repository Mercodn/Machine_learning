When to Use Each Metric

Accuracy

Use when: Classes are balanced and all errors are equally important.

Avoid when: Classes are imbalanced (e.g., 95% no-purchase, 5% purchase).

Precision

Use when: False positives are costly (e.g., spam detection - don't mark legitimate emails as spam).

Focus: Quality of positive predictions.

Recall

Use when: False negatives are costly (e.g., disease detection - don't miss actual cases).

Focus: Completeness of positive predictions.

F1-Score

Use when: You need a single metric balancing both precision and recall.

Best for: Imbalanced datasets where you care about both types of errors.
