Logistic Regression - Variables Used

Input Variables (Features)

The model uses the following features extracted from emails:

  • Email Length: Total number of characters in the email body
  • Number of Exclamation Marks: Count of '!' characters
  • Number of Dollar Signs: Count of '$' characters
  • Number of Uppercase Letters: Count of uppercase letters
Target Variable

Spam Classification: Binary variable indicating whether the email is spam (1) or not spam (0)

Variable Types
  • Numerical Features: All input variables are numerical counts
  • Categorical Target: Binary classification target
