Perceptron - Data Preparation

Data Loading

The Iris dataset is loaded from scikit-learn's datasets module.

Class Selection

Only Setosa (class 0) and Versicolor (class 1) are selected for binary classification, as these classes are linearly separable.

Feature Selection

All four features (sepal length, sepal width, petal length, petal width) are used as input variables.

Train-Test Split

The data is split into training (80%) and testing (20%) sets to evaluate model performance.

Model Training

A Perceptron model is trained on the training data to learn the linear decision boundary between the two iris species.
