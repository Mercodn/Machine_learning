Understanding the Plot

  • Blue Points: Actual data from the training set (actual student grades)
  • Red Line: The regression line (the model's predictions)
  • The Gap: Differences between actual and predicted values are called residuals
  • Best Fit: The line minimizes the total distance of all points from it
